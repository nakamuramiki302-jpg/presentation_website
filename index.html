✨ introduction ✨

こんにちは！私は10-12月の3ヶ月のMRIで、百田先生のもとでプログラミングについて学習しました。
最初はプログラミングについての知識は全くなく、プログラミングは難しそう、自分にできるかな？と思っていました。MRIの期間を経て、現在はアプリ開発までできるようになりました😊
そのプロセスも一緒にお伝えしていきたいと思います🎓

Journey for Three Months 🚀

3ヶ月間の学習の経過をご紹介！ Here's my learning path! 📚

1

Step 1: Udemy講座でAWSのコンセプトを学習 ☁️

Udemyは世界中の講師が作成したさまざまなジャンルの講座を動画形式で購入・受講できるオンライン学習プラットフォームです。動画形式で分かりやすく、自分のペースで学習を進めていきます！

ここではAWS(amazaon web service)の基本概念を学習しました。Deep Learningの仕組みから、EC2やS3などの主要サービスについても理解を深めました。プログラミング学習の第一歩となりました！💪

2

Step 2: Paizaでpythonの講座を受講 🐍

Paizaは実際にコードを書きながら学べるプログラミング学習サイトです。ゲーム感覚で楽しく学習できます！

Pythonの基礎文法から、変数、条件分岐、ループ処理まで学習しました。プログラミングの楽しさを実感し、「コードで問題を解決できる」という感覚を掴めました！😊

3

Step 3: Cytoscapeに着手 🕸️

Cytoscapeは生物学上のパスウェイや分子間相互作用ネットワークなどを可視化・解析できるオープンソースのフリーソフトです。データの関係性を視覚的に表現できます！

本来は生物学的な物質の関連を示す為に用いられますが、今回は筋肉・神経・動脈の関係性を示してみました。これがAnatolink開発の基礎となります🎯

4

Step 4: PaizaでHTML & CSS デザインを学習 🎨

WebページはHTMLで文の構造（見出し、本文など）を作り、CSSで見た目（太字、色など）を整えます。デザインの基礎を学びました！

HTMLタグの使い方、CSSでの色やレイアウトの調整方法を習得。もちろん今はAIが手際よく勝手にcodeを作ってくれますが、codeの意味が分かっているのといないのとでは大きな違いがあります✨

5

Step 5: GitHub & Terminalに挑戦 💻

GitHubは世界中の開発者が使うコードを管理するプラットフォーム。Terminalはコマンドでパソコンを操作するツールです！

初めてTerminalを触り、git add、git commit、git pushなどのコマンドを学習。今もまだTerminalは慣れないですが、少しずつ使いこなせるようになってきたと思います🚀

6

Step 6: AWS Cloud Quest 🎮

AWS Cloud Questはゲーム形式でAWSを学べる学習プラットフォーム。楽しみながら実践的なスキルが身につきます！

ミッションをクリアしながら、EC2インスタンスの起動、S3バケットの作成、VPCの設定などを実践。バッジもGETして達成感を味わいました！クラウドの実践的なスキルを楽しく習得できました🏆
